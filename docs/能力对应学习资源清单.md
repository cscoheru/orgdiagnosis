- **《问题解决力：从混乱到清晰的结构化思维》**  `书籍` `中级`  — 系统讲解MECE、5Why、鱼骨图等工具在真实业务场景中的分层应用，配套每章练习含初级到高级行为对照表，特别适配‘问题定义’与‘结构化分析’二级能力进阶。
- **Coursera: Business Problem Solving with Data Analysis (University of Pennsylvania)**  `在线课程` `中级`  — 聚焦数据驱动的问题识别与验证（如超时率、重复提单率分析），涵盖访谈纪要结构化提炼、A/B测试设计及ROI量化，精准支撑‘问题定义’隐性问题发现与‘价值验证’中级行为。
- **麦肯锡问题分析与解决手册（McKinsey Problem Solving Toolkit）**  `模板` `高级`  — 提供经实战验证的《问题定义说明书》《跨域RACI矩阵》《方案比选结构化矩阵》等12套可直接套用模板，覆盖高级层级所需的多系统协同、四维整合方案与决策支撑要求。
- **IDEO.org Design Kit: Human-Centered Problem Discovery**  `在线课程` `初级`  — 通过视频+交互练习指导一线观察、用户访谈摘要撰写、情绪化表达转结构化纪要等基础技能，直击‘结构化分析’中初级行为的时间戳纪要与隐性诉求提炼要求。
- **The Lean Enterprise Case Library (by Jez Humble et al.)**  `案例库` `高级`  — 收录20+真实跨域转型案例（如ERP-MES-CRM集成断点治理），每个案例含《流程断点热力图》《价值兑现追踪表》及干系人协同纪要，深度匹配‘跨域协同’与‘价值验证’高级行为场景。
- **Miro白板：结构化问题解决工作区（含预置鱼骨图/5Why/RACI/优先级矩阵模板）**  `工具` `中级`  — 支持实时协作绘制根因分析图、跨系统流程泳道图与方案比选矩阵，内置校验逻辑提示MECE完整性，高效支撑‘结构化分析’和‘方案设计’中级独立交付需求。
- **《华为数字化转型之道》第4章：端到端流程断点识别与协同攻坚**  `书籍` `高级`  — 以‘订单到交付’等真实主流程为线索，详解如何拆解战略目标为结构性问题、定位3+业务域断点、输出带前置依赖与干系人的《问题定义说明书》，高度契合高级层级能力画像。
- **APQC Process Classification Framework (PCF) + Cross-Functional Mapping Guide**  `工具` `高级`  — 全球权威流程分类框架，支持快速定位跨供应链/财务/IT等域的流程断点，并生成标准化接口影响范围与系统字段映射表，直接赋能‘问题定义’与‘跨域协同’高级行为中的三系统字段对齐与责任分工。

### 自主学习与适应能力

> 针对模糊业务场景主动探因补缺、解构隐性规则、整合跨域知识，并快速适配新要求推动自动化补缺落地的能力

- **《The Mom Test》**  `书籍` `初级`  — 通过真实对话原则训练用开放式提问挖掘真实需求与隐性动因，完美支撑【主动探因】初级行为中‘3次以上开放式提问’的实践根基。
- **Coursera: 'Business Process Management' (University of Pennsylvania)**  `在线课程` `中级`  — 系统讲授流程建模、断点识别、根因分析及跨系统协同优化方法，直接支撑【模糊解构】【业务补缺】【跨域整合】多个二级能力的独立工作要求。
- **IDEO.org’s Human-Centered Design Toolkit**  `工具` `中级`  — 提供端到端旅程图绘制模板、利益相关方映射画布和隐性断点识别指南，精准匹配【主动探因】高级项中‘4触点3角色5断点’的实战交付需求。
- **McKinsey Problem Solving Toolkit（含Root Cause Tree & Issue Prioritization Matrix）**  `工具` `高级`  — 内置可编辑的根因树框架与优先级矩阵，支持团队级‘问题探因清单’持续迭代与跨团队引用，契合【主动探因】高级行为中结构化归因与知识复用要求。
- **APQC Process Classification Framework (PCF) + Industry Benchmark Reports**  `案例库` `高级`  — 提供覆盖财务、供应链、HR等多领域的标准化流程架构与行业差距数据，是【业务补缺】高级项中定义成熟度模型及开展结构性归因验证的关键对标依据。
- **Miro Business Workflow Templates（含End-to-End Journey Map, RACI Workshop Board, Cross-Functional Swimlane Diagram）**  `模板` `中级`  — 开箱即用的协作式可视化模板，支撑【跨域整合】中级/高级行为中工作坊组织、集成视图绘制与协同动作清单输出，显著降低跨职能对齐门槛。
- **《Seeing What Others Don’t: The Remarkable Ways We Gain Insights》**  `书籍` `高级`  — 深入解析模糊信息中识别模式、发现矛盾目标与构建前瞻性假设的认知机制，为【模糊解构】与【前瞻适配】高级行为提供底层思维模型支撑。
- **Gartner’s ‘Digital Workplace Trend Reports’ + Low-Code/RPA Adoption Playbooks**  `案例库` `高级`  — 聚合全球企业数字化补缺实践、技术适配路径与试点成效数据，直接赋能【前瞻适配】高级项中‘小范围验证性试点’设计与推广可行性判断。
